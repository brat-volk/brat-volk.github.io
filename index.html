MagikIndex is an advanced Win32 keylogger i built to further develop my Windows API and C++ knowledge.
It has been in development for over a year and is now about 2 thousand lines long.
The development has been slow or absent at times, but all things considered i've managed to pump out 11 major updates/releases, all filled with new features.
The most notable features i added to MagikIndex are(as of writing this): Auto-Updating, ZIP Compression, Email Log Delivery with failsafe PS script, ScreenGrabbing with various modes, Context Logging, Admin Privilege abuse, Log Encryption, a really advanced "Trust" system that evaluates the likelyhood of a machine being a sandbox, Detailed Info grabbing, Low-Level Hooking for both mouse and keyboard and finally Clipboard copying.
All these features together make MagikIndex (one of) the best open-source keyloggers on GitHub under many different aspects such as speed, portability, complexity,etc.
This is shown by the amount of people that text me every week for help with MagikIndex despite the short lifespan of the project so far.
MagikIndex also was designed with modularity and ease-of-access in mind in order to help people learn from my code as MagikIndex isn't intend as "Malware" but as research material.
An example of my will to make the code accessible is the "Settings" section in the common.h header.
The Trust system(the anti-debug function basically) is also set up this way, with customizable rules, various security levels and an easily scalable/expandable trust math class.

An upcoming project centered on post-exploiation of comprimised Administrator accounts without using LSASS or LSAKey extraction to fly under the radar.
